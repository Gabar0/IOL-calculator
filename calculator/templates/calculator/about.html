Detailed Explanation of the SRK/T Formula

The SRK/T formula is a sophisticated method used for calculating the power of intraocular lenses (IOLs) for cataract surgery. It combines aspects of geometric optics and regression formulas to predict the effective lens position (ELP) and thus determine the appropriate IOL power. Here is a step-by-step explanation:

1. Retinal Thickness (RETHICK) and AL Value

The formula incorporates retinal thickness, which is dependent on the axial length (AL).

RETHICK is calculated using the formula:
RETHICK = 0.65696 + 0.02029 × AL

2. Modified AL Value (LCOR)

The AL is slightly adjusted to predict the lens position more accurately, particularly for high AL values. This adjustment is referred to as LCOR.

The LCOR is computed using:
LCOR = { AL if AL ≤ 24.2
-3.446 + (1.715 × AL) - (0.0237 × AL²) if AL > 24.2 }

3. Corneal Width (Cw)

The formula uses a transformed AL value (LCOR) and K value (keratometry) to predict the corneal width (Cw).

Cw is calculated as:
Cw = -5.40948 + (0.58412 × LCOR) + (0.098 × K)

4. Optical Calculation (H)

The H parameter is calculated using the R (radius of curvature) and Cw.

If x ≤ 0:
x = R² - (Cw² / 4)

Otherwise:
H = R - √x

5. Effective Lens Position (ELP)

The ACD constant is transformed and combined with the computed H to give the ELP.

ACD constant:
ACD const = H - 3.336
